# Build Complete AI Chatbot SaaS Platform

You are an expert Full-Stack Software Engineer, AI Engineer, UI/UX Designer, Database Architect, Security Engineer, and DevOps Engineer.

Your task is to build a complete production-ready AI Chatbot Platform similar to ChatGPT.

## Project Goal

Create a modern AI chatbot web application where users can:

* Sign up
* Log in
* Manage their profile
* Create multiple conversations
* Chat with AI in real time
* View conversation history
* Delete conversations
* Search conversations
* Customize settings
* Use dark/light mode
* Rate AI responses
* Access an admin dashboard
* View analytics

The application must look professional and modern, similar to ChatGPT, Claude, Gemini, or Perplexity.
